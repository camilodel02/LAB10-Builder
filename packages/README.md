# `packages/`

Aquí van librerías compartidas entre apps (por ejemplo tipos TypeScript comunes, utilidades o un paquete Python publicable).

Por ahora está vacío a propósito: añade subcarpetas con su propio `package.json` o `pyproject.toml` cuando necesites código compartido. Si usas workspaces npm, incluye la nueva carpeta en `workspaces` del `package.json` raíz.
